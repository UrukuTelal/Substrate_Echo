# System Overview

Substrate_Echo is a dissipative, attractor-based dynamical substrate with emergent memory encoding, topology events, and regime-dependent recovery behavior.

## What It Is

A computational architecture for artificial cognition that models information processing as field dynamics in a high-dimensional state space. External information enters as perturbations, not authorities.

## Core Components

```
substrate_echo/
├── core/           # Cognitive modules (memory, planning, meta-cognition)
├── dynamics/       # Field dynamics, attractors, topology, conservation
├── external/       # Epistemic firewall for external agent integration
├── social/         # Persona genomes, relationship memory, dynamics
├── integration/    # Bridges to other systems (PSV, Engine, Council)
├── models/         # Data structures (WorldObject, Experience, Action)
├── scenarios/      # End-to-end simulation scenarios
└── visualization/  # Field and agent rendering
```

## Design Philosophy

**Perturbation, not authority.** External information is a signal source. It enters through an epistemic firewall that evaluates predictive usefulness, not linguistic quality.

**Emergence over specification.** Intelligence is not programmed; it emerges from the interaction of simpler components (attractors, memory, verification, social dynamics).

**Dissipative by nature.** The system does not conserve information. It requires continuous input to maintain coherence. This is a feature, not a bug — it forces the system to engage with reality.
